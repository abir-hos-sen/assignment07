# 🔑 KeenKeeper

Your personal shelf of meaningful connections. Browse, tend, and nurture the relationships that matter most.

---

## 📖 About The Project

KeenKeeper is a relationship management application that helps you stay connected with the people who matter most. Track your interactions, set contact goals, and never lose touch with friends and loved ones.

---

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| **React.js** | Frontend UI framework |
| **React Router DOM** | Page navigation and routing |
| **Recharts** | Data visualization and charts |
| **CSS3** | Custom styling and animations |
| **Context API** | Global state management |
| **Vite** | Build tool and development server |

---

## ⭐ 3 Key Features

### 1. 👫 Friends Management
- View all your friends in a beautiful responsive grid
- Each friend card shows profile picture, name, contact status, and tags
- Click any card to view detailed friend information
- Track days since last contact and set relationship goals

### 2. ⚡ Quick Interaction Logging
- Log calls, texts, and video calls with a single click
- Timeline automatically records every interaction with date and type
- Toast notifications confirm when interactions are logged
- Filter timeline entries by interaction type (Call/Text/Video)

### 3. 📊 Friendship Analytics
- Visual dashboard showing interaction breakdown
- Donut chart displaying percentage of calls, texts, and videos
- Summary cards showing total friends and status overview
- Track which relationships need attention
