# KeenKeeper – Friendship Tracker

A modern web application to help you maintain and nurture meaningful friendships. Track your interactions, set goals, and visualize your social connections.

## ✨ Features

- **Friend Dashboard** – View all friends in a responsive grid with profile pictures, days since last contact, status badges (overdue, almost due, on-track), and tags.
- **Friend Details** – See detailed information, contact goal, next due date, and a progress bar. Use the Quick Check-In buttons (Call, Text, Video) to log interactions.
- **Timeline** – Chronological history of all logged calls, texts, and video chats. Filter by interaction type.
- **Analytics** – Pie chart showing distribution of interaction types (Calls, Texts, Videos) with summary statistics.
- **Responsive Design** – Works seamlessly on mobile, tablet, and desktop.
- **Toast Notifications** – Instant feedback when logging an interaction.
- **404 Page & Loading Animation** – Graceful error handling.

## 🛠️ Technologies Used

- **React 18** – UI library
- **React Router DOM** – Client-side routing
- **Tailwind CSS** – Styling and responsive design
- **Recharts** – Pie chart for analytics
- **React Hot Toast** – Toast notifications
- **Vite** – Build tool and development server
